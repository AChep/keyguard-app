# Keyguard Autofill Lib

This module is largely built with LLMs. It is based on the 20k of crawled web pages
and contains generated data. It is not designed to be easy to fix manually due to that. 
It's mostly a black box or a poor man's neural network.

| Metric | Passes percentage |
|--------|-------------------|
| V1     | 57.0%             |
| V2     | 90.1%             |

Comparing to the V1 implementation I've seen the most improvement in reducing the false-positives.
