On Android, Keyguard plugs into the system autofill framework, so your logins show up right where you sign in — in other apps and in browsers. On Android 11 and newer the suggestions appear inline in your keyboard, so filling a login takes a single tap.

<Screenshot src={autofillKeyboard} caption="An inline autofill suggestion in the keyboard above a login form" />

Suggestions are matched to the app or website you are on, and you can tune how strictly each item matches; see [URL matching](/docs/url-matching/). When nothing matches, a manual option lets you search the vault instead. After filling a login Keyguard can copy its one-time password so it is ready to paste.

Setup takes a minute from Settings → Autofill — see the [autofill & passkeys guide](/docs/autofill/) for the walkthrough, including how to opt in inside Chrome.
