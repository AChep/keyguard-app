Keep every vault open together instead of signing in and out one at a time. You can connect several accounts at once, mixing Bitwarden, self-hosted Bitwarden or Vaultwarden, and KeePass (KDBX) files in the same workspace. Connecting more than one account requires a premium license.

Each account is added through its own secure login and supports two-factor authentication, so the credentials behind each vault stay protected.

<Screenshot src={accountFilter} caption="Two accounts — a Bitwarden vault and a KeePass file — connected together and filterable from the vault list." />

The [multiple accounts guide](/docs/multi-account/) covers the merged vault view, per-account customization, and copying items between accounts.
