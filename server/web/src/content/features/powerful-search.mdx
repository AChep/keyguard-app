Type a few characters and the item you need rises to the top, even when your vault holds thousands of entries. Search matches against the fields inside each item rather than just its title, so a username, an email, or a website fragment is enough to locate the right record.

When several results look alike, narrow them down. You can refine by item type, folder, or tags to cut a broad list into a precise one without leaving the search field.

<Screenshot src={searchFields} caption="A partial query surfaces matches instantly, highlighted across titles, fields, and attachments." />

The aim is simple: surface the exact item with the least typing possible.
